cakephp-preguntafrecuente
=========================

Plugin para ofrecer un sistema de preguntas frecuentes en una página cualquiera.

Las preguntas frecuentes se agrupan en un arbol de categorías.

Cada pregunta frecuente tiene la pregunta como titulo y respuesta.

Utiliza elementos del plugin Utils de CakeDC [https://github.com/CakeDC/utils]
